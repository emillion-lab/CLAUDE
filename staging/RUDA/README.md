# РУДА — Интерактивна карта на минералните находища в България

Single-file, mobile-first, dark theme. Фокус: ☢ уран и 🥇 злато, плюс мед, олово-цинк и други.

**Живо демо (GitHub Pages):** `https://emillion-lab.github.io/RUDA/`

## Данни

| Източник | Обекти | Точност |
|---|---|---|
| USGS MRDS (замразена 2011) | 24 | ±1–2 км, координати от базата |
| МААЕ UDEPO — доклад България | 11 уранови | ≈ по населено място |
| Публични данни (Ада тепе, Милин камък и др.) | 5 | ≈ по населено място |

Урановият добив в България спира 1990–1994 г.; всички уранови обекти са закрити и в рехабилитация (~16 500 tU добити общо, 67 малки находища по данни на Националния геофонд / МААЕ).

## Функции
- Филтър-чипове по суровина (старт: уран + злато)
- Пулсиращи ☢ маркери за урановите обекти
- Попъп с линк към първоизточника (MRDS запис / UDEPO PDF)
- Флаг „≈ координати по населено място" за приблизителните точки
- Без build, без зависимости освен Leaflet CDN

## Правна бележка
Картата е информационна. Проучване/добив изискват разрешение по ЗПБ; проверявай действащите концесии в [НКР](https://nkr.government.bg).
